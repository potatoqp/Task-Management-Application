# Task Management Application

A simple task management application built with Spring Boot and React. It allows users to create and manage task lists and individual tasks, including task status and priority.

## Features
- Create and manage task lists
- Create, update, and view tasks
- Organize tasks with status and priority

## Technologies Used
- Backend: Spring Boot
- Frontend: React
- Database: PostgreSQL running in Docker
